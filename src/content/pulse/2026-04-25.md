### Новости

- [Localization support for web app manifests](https://developer.chrome.com/blog/manifest-localization?hl=en) — Chrome и Edge 148 добавили поддержку локализации для web app manifest. Теперь можно определять многоязычные значения для name, description и других полей прямо в манифесте, используя BCP47 теги.

### Проекты

- [Figma MCP plugin](https://github.com/vkhanhqui/figma-mcp-go) — Бесплатный плагин и MCP для обхода API-лимитов Figma. Позволяет верстать с помощью ИИ без ограничений Rate Limit.
