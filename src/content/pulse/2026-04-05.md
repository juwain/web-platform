#### JS/TS

- [Type guards vs assertion functions](https://www.stefanjudis.com/today-i-learned/the-scope-of-type-guards-and-assertion-functions/) — TypeScript: type guard сужает тип внутри if-блока, assertion function (asserts value is T) — на весь scope после вызова. Полезно при парсинге JSON.
