### Новости

• [TanStack Start rethinks React Server Components](https://react.statuscode.com/issues/470) — TanStack Start получил экспериментальную поддержку RSC. Подход кардинально отличается от Next.js: stream-примитивы вместо «use client», совместимость с React Query и CDN, отказ от Server Actions из соображений безопасности.

### Проекты

• [Tab Out](https://github.com/zarazhangrui/tab-out) — Расширение Chrome заменяет стандартную новую вкладку на группировку по доменам. Лекарство от мультитабоза.
• [Wiretext](https://t.me/front_end_dev/9738) — Unicode wireframe-диаграммы: макеты и схемы собираются из символов. Быстро, текстово, без инструментов.

### Статьи, мнения, туториалы

#### JS/TS

• [AI делает разработку скучной](https://www.marginalia.nu/log/a_132_ai_bores/) — Жёсткий разбор: старые Show HN содержали глубокую проработку, сейчас — AI-генератор поверхностных решений. Prompt ≠ формулировка идеи.
• [One developer, two dozen agents, zero alignment](https://maggieappleton.com/zero-alignment) — Maggie Appleton: запустила два десятка AI-агентов для работы над сайтом. Результат: нулевая согласованность, каждый агент в своём bubble.

#### CSS/HTML

• [Радио стейт-машина в CSS](https://t.me/front_end_dev/9736) — Если UI нужно переключать между визуальными режимами без JS — радио state machine на базе `input type="radio"` и CSS-селекторов. Чистый CSS, без скриптов.
• [Предзагрузка картинок через JavaScript](https://t.me/front_end_dev/9740) — `new Image()` прогревает HTTP cache и показывает картинку без задержки. Когда использовать и когда лучше `link rel="preload"`.

#### Платформа

• [Улучшение UX в legacy-системах](https://www.smashingmagazine.com/2026/04/legacy-systems/) — Практическое руководство: legacy должен сосуществовать с современными продуктами. Big-bang vs incremental vs parallel миграция, фокус на зависимостях и workflow.
• [CSP, CORS и security headers](https://t.me/we_use_js/5278) — Что обязан знать фронтендер: CORS — не про «разрешить запрос», а про контроль доступа. CSP — последняя линия обороны от XSS.
