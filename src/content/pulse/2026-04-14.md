### Новости

• [Name-only @container queries в Safari 26.4](https://webkit.org/blog/17923/name-only-container-queries-a-solution-to-the-naming-wars/) — Контейнеры без size-условия: `@container sidebar { .card { ... } }`. Работает как фильтр для стилей без добавления specificity. Решает проблему BEM-именования и specificity wars. Пока только Safari.

• [MDN rebuilt without React](https://developer.mozilla.org/en-US/blog/mdn-front-end-deep-dive/) — MDN полностью переписан: убрали React, заменили на кастомный render pipeline + Lit web components. Причина: React был обёрткой над статическим контентом, оверхэд не оправдан.

• [WebAIM Million 2026: accessibility ухудшается](https://webaim.org/projects/million/) — Страницы с ARIA имеют в среднем 59.1 ошибок (без ARIA — 42). Больше ARIA ≠ лучше. Состояние a11y регрессирует впервые за годы.

### Статьи, мнения, туториалы

**CSS**

• [Squash and Stretch — принцип Disney в веб-анимациях](https://www.joshwcomeau.com/animation/squash-and-stretch/) — Josh Comeau: первый из 12 принципов анимации Disney для SVG micro-interactions. Пружинная физика, spring easing, hover-эффекты на стрелках. С прицелами на a11y (prefers-reduced-motion).

• [SVG → CSS: конвертер сложных форм](https://css-tip.com/svg-converter/) — Обновлённый конвертер: несколько `<path>` в один `shape()`. Одинарный элемент, отзывчивый код. Версия с `border-shape` для Chrome.

**JS**

• [AI-Generated UI Is Inaccessible by Default](https://frontendmasters.com/blog/ai-generated-ui-is-inaccessible-by-default/) — LLM генерирует визуально правильный `<div onClick>` вместо `<button>`. 10 ошибок a11y в 29 строках. Пятиуровневая система enforcement: prompt constraints → статический анализ → runtime → CI → абстракции.

• [sizes=auto для responsive images](https://www.stefanjudis.com/today-i-learned/intl-can-localize-units-too/) — Chromium уже, Firefox 150 через две недели, WebKit в tech preview. Браузер сам определяет размер, если изображение lazy-loaded. Конец эре мануального `sizes`.

**Мнения**

• [When moving fast, talking is the first thing to break](https://daverupert.com/2026/04/more-talk-less-grok/) — Dave Rupert: при приоритете скорости первым ломается коммуникация между командами. AI усугубляет — «зачем спрашивать эксперта, если LLM всегда скажет да». Дублирующие системы растут, tech debt прячется.
