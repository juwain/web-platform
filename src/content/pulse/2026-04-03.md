### Новости

- [Node.js 25.9: --max-heap-size и stream/iter](https://nodejs.org/en/blog/release/v25.9.0) — Node.js 25.9: флаг --max-heap-size для лимита кучи, экспериментальный stream/iter (ленивые итераторы вместо буферизации), улучшения test runner.
- [EmDash: WordPress-клон от Cloudflare](https://blog.cloudflare.com/emdash-wordpress/) — Cloudflare выпустила EmDash — open-source WordPress-клон на TypeScript/Astro. Плагины sandboxed, нулевое потребление в простое. Минус: нужны платные Cloudflare Workers.

### Статьи, мнения, туториалы

#### AI/Agents

- [Qwen3.6-Plus — SOTA бесплатно](https://qwen.ai/blog?id=qwen3.6) — Alibaba Qwen3.6-Plus — бесплатно на OpenRouter. SWE-bench 78.8% (на уровне Claude 4.5 Sonnet), GPQA 90.4% SOTA. 1М токенов контекста, preserve_thinking для агентных пайплайнов.
- [Three layers of system health](https://arxiv.org/abs/2603.22106) — Margaret-Anne Storey: три слоя долга системы. Technical debt — в коде, cognitive debt — в людях, intent debt — в артефактах. LLM-код усугубляет когнитивный и целевой долг.

#### JS/TS

- [What To Know in JavaScript (2026 Edition)](https://frontendmasters.com/blog/what-to-know-in-javascript-2026-edition/) — Chris Coyier: полный обзор JS на 2026 — Iterator Helpers, Set Methods, RegExp.escape(), Promise.try(), Import Attributes (JSON + CSS), Temporal API. С примерами и CodePen-демками.

#### CSS

- [view-transitions-toolkit от Chrome Labs](https://chrome.dev/view-transitions-toolkit/) — Chrome Labs выпустили view-transitions-toolkit — утилиты для View Transitions: оптимизация keyframes, пауза/скраббинг, auto navigation types, feature detection.
- [CSS subgrid is super good](https://dbushell.com/2026/04/02/css-subgrid-is-super-good/) — CSS subgrid для CMS-контента: named grid lines, breakout full-width блоков одним классом, вложенные subgrid-уровни. Забудьте про negative margins.
- [Sliding liquid oozing reveal effect](https://css-tip.com/sliding-liquid/) — CSS shape() для liquid-эффекта reveal — clip-path-анимация между двумя wavy-формами. Генератор форм на css-generators.com.

#### HTML/Accessibility

- [Stay Away From Accessibility Overlays](https://vale.rocks/posts/accessibility-overlays) — Accessibility overlays вредят больше чем помогают: не фиксируют реальные проблемы, мешают скринридерам, привлекают суды. FTC оштрафовала accessiBe на $1M. Не используйте.
- [loading="lazy" для video и audio](https://engineering.squarespace.com/blog/2026/squarespace-and-web-standards-how-we-helped-bring-html-video-and-audio-lazy-loading-to-todays-browsers) — Squarespace протолкнули loading="lazy" для <video> и <audio> в WHATWG. Стандарт с 23 марта 2026, работает во всех браузерах.
- [Chrome 146: meta name="text-scale"](https://developer.chrome.com/release-notes/146) — Chrome 146: <meta name="text-scale" content="scale" /> — заставляет rem/em учитывать системный размер шрифта пользователя. Решает проблему мобильных браузеров, игнорирующих настройки.

#### Инструменты

- [Keep Pull Requests Small](https://www.frontendjoy.com/p/keep-pull-requests-small-junior-dev) — 3000+ PR: большие получают поверхностные ревью. AI-код прячет баги в больших PR. Параллельные ревью, меньше merge conflicts, растут архитектурные навыки.
