### Новости

• [React 20 stable](https://react.dev/blog/2026/04/09/react-20) — React 20 stable: compiler включён по умолчанию, новый reconciler с приоритетом пользовательского ввода, server actions 2x быстрее. Миграция с React 19 — пара часов.

• [Rspack 2.0](https://rspack.dev/blog/rspack-2) — Rspack 2.0: полный переход на SWC, билды на 40% быстрее, incremental compilation, улучшенная совместимость с Webpack 5 plugin API.

• [StackBlitz 5: WebContainer API](https://blog.stackblitz.com/posts/stackblitz-5/) — StackBlitz 5 открывает WebContainer API для третьих лиц. Full Node.js в браузере для демо, документации, онлайн-IDE. Бесплатно до 10k MAU.

### AI/Agents

• [AI Tools That Actually Help](https://simonwillison.net/2026/Apr/9/ai-tools-that-actually-help/) — Simon Willison: в 2026 AI-инструменты для разработчиков делятся на «быстрый старт» и «долгое сопротивление». Первые дают 2x на первой неделе, вторые — 10x через 6 месяцев.

• [The code review is the bottleneck](https://zainrav.com/blog/ai-code-review-bottleneck) — Zain Ravi: AI пишет код в 10x быстрее, но code review — тот же. Объём PR растёт, качество ревью падает. Решение: договориться, что AI-код имеет свои стандарты доверия.

• [Llama 4 Maverick: 256-модель ensemble](https://ai.meta.com/blog/llama-4-maverick/) — Meta выпустила Llama 4 Maverick: 256 специализированных моделей в ensemble, routing через MoE. 128k контекст.(weights открыты).

### JS/TS

• [React Server Components в production](https://vercel.com/blog/react-server-components-in-production) — Vercel: RSC в production — что работает, что нет. Streaming — да,Suspense boundaries — критичны, client components для интерактива — неизбежны. Честный разбор без хайпа.

• [Jest → Vitest: реальный опыт миграции](https://t.me/front_everyday/1286) — Разбор миграции большого монорепо: моки ведут себя иначе, environment variables через vi.stubEnv, barrel exports тормозят. Не drop-in замена, но результат стоит effort.

### CSS

• [border-shape: anything is a box](https://css-tip.com/border-shape/) — border-shape в Chrome 147: элемент может быть треугольником, звездой, blob — и при этом иметь тени, рамки, outline. clip-path режет, border-shape формирует. Полная совместимость с border-radius.

• [CSS-only parallax](https://css-tip.com/parallax/) — Параллакс на чистом CSS: perspective на контейнере, translateZ на слоях, scroll-driven для автоматического движения. Zero JavaScript.

### Performance

• [The cost of Web Components](https://component-odyssey.com/posts/the-cost-of-web-components/) — Бенчмарк: Shadow DOM adds ~0.5ms per element creation. CSS-in-shadow isolation prevents style recalc. 10k components: 5ms overhead. Custom Elements v1 upgrade callback — free performance win.
