### Новости

- [TypeScript 7.0: компилятор на Go, 10x быстрее](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0-beta/) — TypeScript переезжает на Go, получая колоссальный прирост производительности — в 10 раз быстрее, чем TypeScript 6.0. Парсинг, типизация и эмиссия теперь работают параллельно, что кардинально ускорит большие проекты.
- [BCD Changes Report: Serial API, LanguageModel, CSS Container Rule](https://bcd-watch.igalia.com/weekly/2026-04-27.html) — Firefox добавил поддержку Serial API для работы с последовательными портами и LanguageModel API. В CSS появился Container Rule и свойство position-anchor. Всего за неделю +40 новых API в браузерах.

### Проекты

- [TSRX: духовный преемник JSX для TypeScript](https://tsrx.dev/) — Языковое расширение TypeScript от Dominic Gannaway, которое позиционируется как духовный преемник JSX. Позволяет писать более декларативный код с улучшенной типизацией шаблонных конструкций.
- [Rspack 2.0: much faster, better ESM support](https://rspack.rs/blog/announcing-2-0) — Rspack обновился до версии 2.0 с существенным улучшением производительности и поддержкой ES modules. Бандлер продолжает расти популярностью как альтернатива Webpack.

### Статьи, мнения, туториалы

#### JS/TS

- [collective speed isn't the sum of individual speed](https://blog.jim-nielsen.com/2026/collective-speed-isnt-the-sum-of individual-speed/) — Глубокая мысль: быстрые разработчики не делают быструю компанию, как спринтеры в эстафете. Критически важна передача эстафеты (синхронизация) между участниками, а не индивидуальная скорость.
- [AI & alignment: raw coding speed isn't the bottleneck](https://chriscoyier.net/2026/04/25/ai-alignment/) — Хорошая статья о том, что скорость написания кода не является основной проблемой. Важнее синхронизация команды, понимание того, что строить, и review процесс. AI ускоряет индивидуальную работу, но не помогает с командной.

#### CSS

- [Rebuilding a web text editor: draft.js is not the answer](https://dev.to/readymag/rebuilding-a-web-text-editor-3g2o) — Readymag CTO о том, почему они отказались от draft.js и решили строить свой текстовый редактор с нуля. Интересный кейс про vendor lock, emoji sequences и необходимость full control над текстовыми манипуляциями.

#### Инструменты

- [Apify Actors для OpenClaw: 25,000+ инструментов для веб-данных](https://apify.com/integrations/openclaw) — Интеграция Apify в OpenClaw дает доступ к более чем 25,000 инструментам для реального сбора веб-данных. Один CLI install для расширенных возможностей парсинга и извлечения данных.
- [WSL9x: Linux kernel внутри Windows 9x kernel](https://codeberg.org/hails/wsl9x) — Любопытный проект: современный Linux kernel, работающий кооперативно внутри ядра Windows 9x. Технически невозможное стало возможным.
