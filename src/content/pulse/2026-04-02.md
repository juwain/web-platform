### Новости

- [RedwoodSDK 1.0](https://rwsdk.com/blog/redwood-v1-getting-out-of-the-weeds) — RedwoodSDK 1.0 — минимальный React SDK на Cloudflare: без code generation, RSC из коробки, useSyncedState для realtime через Durable Objects. Workers, D1, R2, Queues — first-class.
- [Next.js Adapters API](https://nextjs.org/blog/nextjs-across-platforms) — Next.js 16.2: стабильный Adapter API для деплоя на любую платформу — OpenNext, Netlify, Cloudflare, AWS Amplify. Публичный test suite для валидации адаптеров.
- [Babylon.js 9.0](https://www.babylonjs.com/) — Babylon.js 9.0 от Microsoft: clustered lighting, node-based particle editor, volumetric lighting, Gaussian Splatting. Год новых фич и оптимизаций.

#### AI/Agents

- [Your ticket is a prompt](https://dheer.co/tickets-are-prompts/) — Тикеты теперь промпты для агентов. Фрагментированные тикеты = фрагментированный код. Давайте агентам максимальный контекст — пусть сами делят на подзадачи.
- [Junior Devs in the Age of AI](https://www.frontendjoy.com/p/junior-devs-in-the-age-of-ai) — Быть junior'ом стало проще (AI как ментор) и сложнее (harder to hire). Soft skills — новые hard skills. Разница junior/mid сокращается.
- [Ban Ray: Your face is not inventory](https://banray.eu/en/index.html) — Ban Ray — кампании против Meta Ray-Ban: 7M очков с камерами, видео отправляют в Кению для ревью, Name Tag для facial recognition. GDPR-нарушения, class action.

- [Inertia 3.0](https://laravel.com/blog/inertia-v3-whats-changed-since-the-first-beta) — Inertia 3.0 — Vite 8 plugin, SSR в dev-режиме, optimistic updates. Интеграция React/Vue/Svelte с серверными роутерами.
