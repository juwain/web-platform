**👾 Пульс веб-платформы 01.04.2026**

### Новости

- [JetStream 3: новый бенчмарк для веб-приложений](https://blog.chromium.org/2026/03/jetstream-3-a-modern-benchmark.html) — JetStream 3 — совместный бенчмарк Google, Apple, Mozilla. Wasm теперь 15-20% тестов (SIMD, WasmGC, exceptions). Новые workload'ы: Transformers.js, SQLite3, Dart/Kotlin UI, .NET runtime.
- [Supply chain-атака на Axios](https://javascriptweekly.com/issues/779) — Axios 1.14.1 и 0.30.4 — malicious dependency через postinstall. 100M+ загрузок/неделю. Зафиксируйте версии, проверьте npm install за сегодня.
- [Утекли исходники Claude Code](https://habr.com/ru/companies/bar/articles/1017574/) — Исходники Claude Code утекли через sourcemap в npm — 785 KB main.tsx, 40+ инструментов, система «снов», Buddy-компаньон, KAIROS (всегда включённый Claude), ULTRAPLAN (30-минутное планирование), режим прикрытия для Anthropic-сотрудников.
- [Navigation API — кросс-браузерный](https://web.dev/blog/baseline-navigation-api) — Navigation API — Baseline, кросс-браузерный. event.intercept для навигации, адресная строка как источник состояния. Замена History API.

### Статьи, мнения, туториалы

#### AI/Agents

- [Encoding Team Standards](https://martinfowler.com/articles/reduce-friction-ai/encoding-team-standards.html) — Стандарты команды как versioned AI-instructions — versioned, reviewed, shared. Качество кода больше не зависит от того, кто промптит.
- [Claude нашёл 14 уязвимостей в Firefox](https://blog.mozilla.org/en/firefox/hardening-firefox-anthropic-red-team/) — Anthropic: Claude проанализировал исходники Firefox и нашёл 14 критических уязвимостей. LLM начинают реально применяться в security-аудитах.
- [Webpage MCP — сайт как MCP-сервер](https://github.com/mcpland/webpage-mcp) — Webpage MCP — сайт как MCP-сервер для управления агентами: навигация, скриншоты, клики, чтение контента. Native Messaging, кросс-вкладки.

#### JS/TS

- [Port React Compiler to Rust](https://github.com/facebook/react/pull/36173) — React Compiler портируют на Rust — Babel plugin, OXC/SWC crates. 1717/1718 тестов проходят. Ну и куда без Claude...
- [Vue 3.5 useId()](https://t.me/front_everyday/1285) — Vue 3.5 useId() — детерминированный уникальный ID для компонентов. Решает hydration mismatch в Nuxt, без Math.random().

#### CSS

- [CSS-only Chrome Dino Game](https://css-tip.com/dino-game/) — Chrome Dino на чистом CSS — jump/crouch клавишами. Scroll-driven animations, @property, container queries. Chrome-only.

#### HTML/Accessibility

- [focusgroup — нативная клавиатурная навигация](https://blogs.windows.com/msedgedev/2026/03/05/making-keyboard-navigation-effortless/) — HTML-атрибут focusgroup для клавиатурной навигации — стрелки, skip disabled, restore focus без JS. Edge/Chromium.
- [Test IDs are an a11y smell](https://tkdodo.eu/blog/test-ids-are-an-a11y-smell) — data-testid — a11y smell. Тесты должны искать элементы как пользователи и скринридеры, а не по тестовым ID.
