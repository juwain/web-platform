### Новости

• [No AI in Node.js Core](https://cekrem.github.io/posts/no-ai-in-nodejs-core/) — Петиция против AI-сгенерированного кода в ядре Node.js. Повод: PR на 19k строк, написанный Claude Code. Аргументы: code review AI-кода не компаундирует, LLM не учится на фидбеке, open-source не должен требовать платной подписки для воспроизведения.

• [React 19.2.5: патч RSC-уязвимости](https://github.com/facebook/react/releases/tag/v19.2.5) — Срочный релиз React 19.0.5, 19.1.6 и 19.2.5 закрывает уязвимость в React Server Components. Обновление обязательно.

### Статьи, мнения, туториалы

**JS/TS**

• [React: оптимизация diff view на GitHub](https://react.statuscode.com/issues/469) — Разбор от команды GitHub: как урезали потребление памяти и задержки в React-компоненте PR diff. Глубина дерева, useEffect, спрей обработчиков — практические выводы.

• [Signals: push-pull алгоритм](https://willybrauner.com/journal/signal-the-push-pull-based-algorithm) — Подробное объяснение работы signals изнутри: как в Solid, Vue, Preact и Angular. Сравнение с React-подходом.

• [ESLint → Oxlint: миграция](https://blog.logrocket.com/retire-eslint-migrate-oxlint/) — Oxlint на Rust: в 50-100x быстрее ESLint, type-aware linting из коробки, меньше конфигурации. Когда имеет смысл мигрировать и что теряете.

• [Нативные JSON-модули](https://allthingssmitty.com/2026/03/16/native-json-modules-are-finally-real/) — `import config from "./config.json" with { type: "json" }` работает в runtime с ESM-кэшем. Больше не нужны loader'ы и трансформации для JSON.

• [Три причины раздувания зависимостей в JS](https://43081j.com/2026/03/three-pillars-of-javascript-bloat) — Поддержка древних рантаймов, мутации глобалов, кросс-realm значения — три источника dependency bloat. Где заменить зависимости нативными API.

**CSS**

• [Инвертированные темы через light-dark()](https://daverupert.com/2026/04/inverted-light-dark/) — Дейв Руперт: `data-theme="inverted"` + `light-dark()` для автоматического переключения цветовой схемы. Без дублирования стилей, один файл.

**HTML**

• [focusgroup: декларативная навигация стрелками](https://developer.chrome.com/blog/focusgroup-rfc) — Атрибут `focusgroup` для toolbar, tablist, menu — стрелочная навигация без ручного tabindex. Направление и восстановление фокуса на платформе.

**Инструменты**

• [Ladybird: 352 PR за март, YouTube 4K](https://ladybird.org/newsletter/2026-03-31/) — Независимый браузер продвигается: AsmInt-интерпретатор для JS (Kraken +44%, SunSpider +69%), VP9/Opus для MSE, выбор качества на YouTube до 4K.

• [Railway уходит с Next.js](https://blog.railway.com/p/moving-railways-frontend-off-nextjs) — Миграция фронтенда Railway с Next.js на Vite + TanStack Router. Причины и опыт.
